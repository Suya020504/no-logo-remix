## B. 기획서 문안 반영 내역 (v2 → v2.1, 총 4곳)

### ① 기획내용 첫 문단 — 배경 근거 완성 (이해도 20점)

- **전(v2)**: "실제로 `[출처 확인 후 기입]`이며 … `[출처 확인]`, 문제는 전시회 자체가 아니라…" (빈 슬롯 2개)
- **후(v2.1)**: 진흥회 통계(736건·953만 명 + **연령 집계 부재**) → 팝업 대비(81.4%·73.0%) → **불교박람회 선례(25만 명·2030 73%·사전등록 4배)** → "문제는 '역할'이 없다는 것" 4단 논증으로 완성
- 특히 "공식 통계에 2030 데이터가 없다"는 사실은 약점이 아니라 **우리 '관심 신호 리포트'의 존재 이유**로 역이용했고, 불교박람회는 "전시회도 핫플이 될 수 있다"는 주제의 실증 선례로 배치

### ② 추진방안 '사전 단계' — 노쇼 상쇄 논리 추가

- **후**: "토큰 선지급은 '현장에서만 받을 수 있는 보상'으로, 사전등록의 고질적 약점인 노쇼를 상쇄하는 장치다." (노쇼 수치는 해외·온라인 행사 기준이라 지면에는 수치 없이 논리만 — 정확성 원칙)

### ③ 기대효과 KPI 문단 — 벤치마크 + 실행력 논리 (기대효과 15점)

- **후**: 서울교통공사 스탬프투어 공공 사례(이용객 2만여 명 증대, 응답자 87% 만족) + **"여정 전체가 구동되는 프로토타입을 제작·배포했고, KPI는 그 안에 측정 체계로 설계되어 파일럿 첫날부터 실측 가능"**
- ※ 당초 '자체 파일럿 실측치 기입' 방식은 폐기 — 아이디어 제안 단계에서 실측을 전제하면 채울 수 없는 빈칸(=거짓 위험)이 됨. 사실만으로 구성: 프로토타입 존재(사실) + 측정 설계 존재(사실, 분석 이벤트 19종) + 실측은 파일럿 운영의 몫으로 정직하게 이관

### ④ 편집 체크리스트 갱신

- 출처 조사 항목 완료 처리. **남은 빈칸은 파일럿 실측치(N·완주율·소요시간)뿐**

## D. 인용 시 주의사항 (검증 과정에서 나온 표현 정확성 메모)

1. "약 4배"(불교박람회 사전등록)는 산술 계산값(44,365÷11,187≈3.97) — 원문 문장에 '4배' 표현은 없음. 현재 문안의 "약 4배" 표기는 산술적으로 타당하나, 심사 Q&A 시 "2023년 1만1187명 → 2026년 4만4365명"으로 답할 것
2. 서울교통공사 만족도는 "응답자의 87%가 만족" 표현이 원문과 일치 (현 문안 반영됨)
3. 트렌드모니터 "20대 89.1%"는 방문률이 아니라 '주 방문 연령대 인식' — 방문률로 쓰면 오인용. 방문률은 81.4%(전체)·82.8%(2023년 20대)를 사용할 것
4. 진흥회 통계의 유형별 수치(무역 181.7만 등)에서 12,794 등은 '전시회당 평균'이고 1,816,756이 '합계' — 혼용 주의
5. 모든 원문 URL은 A절 표 참조 — 본선 발표 자료 만들 때 재확인 가능
