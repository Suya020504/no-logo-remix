## ⑥ 발견 → 수정 → 재검증

1. **[결함→수정] 커스텀 미션 추천 변별력 상실** — 배터리 전시에선 전 카드에 '배터리'가 들어가 첫 일치 중단 매칭이 전 카드 동점을 만듦 (강현 주행에서 ESS 카드가 top3 밖). → 일치 키워드 누적 + 불용어('없는·걱정' 등) 제외로 수정, 재검증에서 ESS 카드 1순위 복귀. **수직 전시 실데이터로 주행했기에 잡힌 결함.**
2. [관찰] 결과명-미션 정합성 한계 — 미션 카테고리가 조합에 없으면 기술 기준 이름. '제목 1회 수정'이 안전판으로 작동(하람). 근본 개선은 백로그.
3. [백로그] 스타일 락 이미지 5종의 도메인 커버리지 — 전시 도메인별 세트 사전 생성 운영 절차 필요.
4. [백로그] B2B 관람객 톤 미스매치 — 강현 ★3(6인 중 최저). '비즈니스 모드' 문구 톤 옵션. 반대로 리드 이원화(도입상담/채용 분리)는 B2B 페르소나 2인 모두 즉시 사용 — 설계 유효 확인.
