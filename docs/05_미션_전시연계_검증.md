## 🎯 보너스 — 송도컨벤시아 개최 전시와의 겹침 (본선 어필 포인트)

- **그린에너텍** (송도컨벤시아, 매년 8월, 인천시 주최) — 청정에너지·탄소중립·**친환경 플라스틱** → m1·m3 직결
- **대한민국 드론·UAM 박람회** (송도컨벤시아, 매년 7월, 국토부·인천시 주최) → m4 직결

→ 본선(송도컨벤시아 개최)에서 "이 미션들은 바로 이 전시장에서 열리는 실제 전시에 그대로 적용 가능합니다"라고 말할 수 있음.

## 리밸런싱 판단 요약

- 기존 5개 미션은 전부 실존 전시 분야와 연결 확인 → **유지**
- 카테고리 '건강·의료'에 대응 미션이 없던 공백 → **m6 신설** (KIMES 연계). 신설 후 추천 알고리즘 검증: m6 선택 시 케어웨이브(No.5)·더마픽(No.24) 헬스 기술이 추천 상위 노출 확인 ✅
- 로봇은 최근 전시산업 최대 화두(로보월드 20주년 역대 최대) → **m7 신설**
- 미션 카드에 전시 연계 태그를 노출해 "생활문제 ↔ 산업 전시"의 연결을 관람객에게도 보여줌
