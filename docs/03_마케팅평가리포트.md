## 1. 종합 판정

**아이디어는 본선급, 제출물의 '정체성'이 리스크다.**

블라인드 선택→리빌→QR 부스 방문→AI 리믹스로 이어지는 전환 사슬의 독창성, 동작하는 배포 프로토타입, KPI 산식 고정·'가상 예시' 표기 같은 실행력 증거는 5개 관점 모두가 "대학생 출품작 기준 최상위"로 평가했다. 그러나 공고가 요구한 것은 **"전시장 밖 2030을 유입시키는 마케팅 전략"**인데, 현재 자료는 **"이미 입장한 관람객의 경험 플랫폼 기능명세서"**로 읽힌다. 이 프레이밍 문제 하나가 이해도(20점)·문제해결(15점)·표현력(15점)에서 동시 감점을 만든다.

| 구분 | 판정 |
|---|---|
| **현재 상태 제출 시** | **60~65점 / 100** — 본선 진출 확률 40~50% (경계선) |
| **P0 액션 반영 시** | **75~80점** — 본선 진출 유력 |
| 결론 | 아이디어 결함이 아닌 **프레이밍·서술 문제 → 개선 탄력성 큼** |

## 4. 전 관점 공통 강점 (기획서에서 반드시 팔아야 할 것)

1. **전환 설계의 논리 사슬** — "블라인드가 관심을 만들고, 공개·QR이 방문을 만들고, 토큰·AI가 공유 콘텐츠를 만든다" (전 관점 일치)
2. **선택이 브랜드 공개 전에 일어나는 유일한 구조** — 중소기업이 대기업과 동일 조건에서 첫 평가를 받는다 (Hidden Champion 서사)
3. **동작하는 배포 프로토타입** + 화면 12종·상태 13종·QA 기준·이벤트 19종 — 경쟁작이 흉내 내기 어려운 실행력 증거
4. **자기방어 장치 내장** — KPI 산식 고정, '가상 예시' 표기, 옵트인 기본 OFF, AI 면책 문구 → 숫자 부풀리기 공격 선제 차단
5. **공유를 사후 기능이 아닌 산출물 규격으로 설계** — 9:16 카드 = 전광판 = 모바일 동일 정보구조

## 5. 핵심 약점 (복수 관점 반복 지적)

1. **[최대] 장외 유입 전략 부재** — 모든 여정이 행사장 QR에서 시작. 사전 티저·채널 전략·사전등록 퍼널·공유→신규 유입 환류가 0개 (3개 관점 공통)
2. **제출물이 마케팅 전략서가 아님** — 문제→인사이트→전략→효과 스토리라인과 배경 논증 전무
3. **대기업 블라인드 반발 미해소** — 반대급부 설계가 한 문장뿐 (4개 관점 공통)
4. **운영 비용-편익·실행 주체 공백** — 기업 섭외, AI 단가, 검수 인력, 예산 없음
5. **가상 KPI의 거짓 정밀성** — '38.7%' 류 수치는 근거를 물으면 답이 없음
6. **체험 소요시간 과소 표기** — 표기 10~15분 vs 실제 부스 2~3곳 이동 시 40분+, 토큰 1개 시점 이탈 방어선 부재

## 6. 기획서 제출 전 개선 액션 (배점 영향 순)

### P0 — 필수 (이것 없이 제출하면 경계선)

| 액션 | 해결하는 것 |
|---|---|
| **① 기획서를 '3단 마케팅 퍼널'로 전면 재작성**: 전시 전(블라인드 카드 SNS 티저 캠페인) → 전시 중(NO LOGO REMIX 체험) → 전시 후(UGC 재확산). 화면 상세·개발 어휘(P01~P10, [M]/[S])는 부록·QR로 강등 | 이해도 20 + 표현력 15 — 최대 배점 영향 단일 액션 |
| **② 장외 유입 장치 신설**: 행사 1~2주 전 인스타·틱톡 블라인드 카드 퀴즈("이 기술의 주인은 대기업일까 스타트업일까?") + 정답자 토큰 선지급·사전등록 딥링크. 공유 카드 수신자용 랜딩(원격 투표 + "너도 만들러 가기") | 문제해결 15 — 공고 핵심 요구 직접 대응 |
| **③ 배경·필요성 섹션 신설**: 2030 전시회 인식 통계 2~3개, 유사 포맷 벤치마킹(복면가왕·블라인드 테이스팅·기존 스탬프랠리 참여율), '왜 블라인드인가' 논증 1~2p | 이해도 20 — 논증으로만 채점되는 항목, 현재 완전 공백 |
| **④ 가상 KPI → 벤치마크 기반 목표치 전환**: 정밀 가상 수치 삭제, "유사 QR 랠리 참여율 X%(출처) 대비 목표 Y%" 형식 | 기대효과 15 |

### P1 — 강력 권장

- **기업 섭외 논리 이원화**: 대기업용("리빌 서프라이즈 캠페인" — 브랜드 후광 제거된 순수 기술 선호 데이터) / 중소기업용(Hidden Champion 노출) + 지표 번역표(블라인드 선택 수→순수 기술 관심, 자료 요청 수→검증된 옵트인 리드) + 1개 홀 20부스 파일럿 규모 명시
- **운영 비용-편익 1페이지**: AI 생성 단가·검수 인력·QR 제작비 추정 vs 체류시간 증가·재계약율·리포트 유료화 편익
- **차별점 비교표 1장**: "스탬프는 방문의 증거로 끝나지만, 토큰은 창작 재료가 된다" / "선택이 브랜드 공개 전에 일어난다" — 환원 공격("스탬프투어+AI포토부스 조합 아닌가") 차단
- **생성형 AI 활용 범위 명시 문구를 제출물 표지에 포함** (대회 규정 필수 — 이미 준비된 문구를 누락하지 말 것)

### P2 — 여유 시

- 체험 소요시간 2트랙 재표기: "핵심 체험 15분(부스 1곳 라이트 리믹스) + 확장 트랙(3토큰 풀 리믹스)" + 토큰 1개 라이트 모드
- 브랜드 정리: 슬로건 "브랜드는 숨기고, 기술을 발견하세요" 1문장 통일 / 해시태그 #NOLOGOREMIX·#노로고리믹스·아이디어명 자동 태그로 교체(범용 태그 삭제) / MIXI 컬러 단일화 + 상표 검토(일본 SNS mixi 동명)
- 화제성 현장 연출: 매시 정각 전광판 단체 리빌 "REVEAL O'CLOCK" + 포토존 + "지금 전광판에 떴어요" 알림
