- **+0~1h — 전원:** 주제 분해(핵심 요구 3개 이내로), **예선 주제와의 연속성 여부 판정(연속이면 C등급 카드 즉시 부활)**, 당일 공개 심사기준을 셀프 채점표에 재매핑, 자산 맵 대조(쓸 카드 O/X)
- **+1~2.5h — 전원:** 아이디어 발산 → '한 문장 환원 공격' 테스트로 컷 → 컨셉 1개 확정
- **+2.5h~ — 병렬 분기:** 김주희 골격 초안 / 이연수 근거 카드·KPI / 최현규 프로토타입 데이터 교체 착수 / 차서빈 템플릿에 컨셉 이식
- **저녁 — 교차 리뷰 1차:** 결함 카탈로그 6종으로 상호 공격, 예상 질문 후보 수집

- **정오 — 교차 리뷰 2차:** 셀프 채점 → 복수 지적 항목만 수정 (개인 취향 지적은 버림)
- **오후:** 분량 압축·디자인 마감·AI 활용 명시 작성(§3-5) → **마감 1시간 전 제출 목표 (버퍼 고정)**
- **제출 후:** 발표 리허설 2회, 예상 질문 1순위를 뽑아 답변 스크립트화 → 8/14 당일 체크리스트(§1(e))로 연결

### 3-3. PPT 골격 (주제 무관 목차)
1. **표지·요약** — 첫 문장에서 문서의 정체(요구된 문서 유형)를 선언
2. **문제정의** — 주제의 핵심 요구를 직접 지목
3. **근거** — 검증 카드만 (출처 못 찾으면 수치 빼고 논증 문장으로)
4. **아이디어** — 차별점 한 문장("기존 X는 ~로 끝나지만, 본 기획은 ~") + 비교표
5. **실행** — 주체·비용·일정 + 이해관계자 이원 설득 논리
6. **효과** — 3주체 가치 (+ 개최지 연계 문장 검토 — 인천관광공사장상 앵글, 예선 송도 문장 부활 지점)
7. **KPI** — 3부 템플릿(지표 제안 → 검증 사례 인용 → 산출물 내 측정 체계). '가상 예시' 표기
8. **AI 활용 명시 + 부록** — 기술 상세는 여기로 강등

### 3-4. 셀프 리뷰 체크리스트 (제출 전 전원 통과 필수)
- [ ] 당일 주제의 핵심 요구를 **직접** 푸는 장치가 본문에 있는가
- [ ] "기존 X의 조합 아닌가" 환원 공격에 답하는 차별점 문장/비교표가 있는가
- [ ] 산출물이 요구된 문서 유형으로 읽히는가 (기능명세서 어투 금지)
- [ ] 없는 출처 0건 — 출처 없으면 수치 삭제, 논증만 유지했는가
- [ ] 실행 주체·비용·일정 + 이해관계자 반대급부가 있는가
- [ ] 소요시간 등 운영 표기값이 현실과 어긋나지 않는가
- [ ] 가상 정밀 수치('38.7%' 류) 0건 — "검증 사례(출처) 대비 목표" 형식인가, 자체 실측 전제 0건인가
- [ ] 주장마다 자기방어 장치('가상 예시' 표기·산식·면책)가 붙었는가
- [ ] 문제→인사이트→전략→효과 흐름, 개발 어휘는 부록인가
- [ ] 중복 서술 몰아주기·고유명사 축약·분량 압축 완료인가
- [ ] 예상 질문 1순위 1건을 본문에서 선제 답변했는가
- [ ] **AI 활용 명시 문구가 실제 사용 내역과 일치하는가 — 사용하지 않은 항목 삭제, 사용한 항목 누락 없음을 전원 확인** / 데모·QR은 검증 통로로만 배치했는가
- [ ] 오인용 주의 5건(§2-1) 위반 0건인가

### 3-5. AI 활용 명시 — 당일 작성 절차 (당일 안내 양식이 있으면 그 양식 우선)
미리 채운 문안을 붙여넣지 않는다. **실제 사용한 항목에만 체크하고, 체크된 항목만 문장으로 옮긴다** (공고문: 제출서류 내용이 사실과 다를 경우 수상 취소 등 조치).

- ☐ 문안 초안 작성·윤문에 사용했다 → "문안 초안 작성 및 윤문"
- ☐ 자료 정리 보조에 사용했다 → "자료 정리 보조"
- ☐ 프로토타입 코드 작성 보조에 사용했다 → "프로토타입 코드 작성 보조"
- ☐ 이미지·디자인 시안 생성에 사용했다 → "디자인 시안 생성"
- ☐ 기타 실제 사용 내역: ______
- 마무리 고정 문장: "아이디어 도출, 인용 자료의 출처 확인, 최종 의사결정은 팀원이 직접 수행하였습니다."
- 기획 내용 자체에 AI 기능이 포함될 경우: "서비스 내 AI 기능은 기획 대상으로서 본문에 별도 기술하였으며, 위 작성 과정의 AI 활용과 구분됩니다." 추가
